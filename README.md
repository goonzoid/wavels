## usage

```
wavels 0.1.0

USAGE:
    wavels [flags] [wav_file|directory ...]

FLAGS:
    -c, --count           show counts, grouped by sample rate, bit depth, and channel count
    -r, --recurse         recursively list subdirectories
    -h, --help            show this help info
    -v, --version         show version info
```
